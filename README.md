# Conversor de Monedas - Challenge Alura

Aplicación de consola en Java que convierte monedas en tiempo real usando una API externa. Desarrollada como challenge del programa Oracle Next Education (Alura).

**Autor:** William Esteban Moya Galvis

## Características principales

- Menú interactivo en consola con opciones de conversión
- Conversión en tiempo real con API (ExchangeRate-API)
- Soporte para múltiples monedas (USD, EUR, COP, entre otras)
- Manejo básico de errores (conexión, respuesta inválida)
- Uso de records para datos inmutables




## Tecnologías utilizadas

- Java 25 (JDK nativo)
- HttpClient (JDK nativo para peticiones HTTP)
- Gson (para parseo de JSON)
- IntelliJ IDEA (entorno de desarrollo)
- Records (Java 16+ para datos inmutables)
